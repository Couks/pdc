# Guia para Rodar o Projeto Whats Your Finances

Siga os passos abaixo para garantir que tudo funcione corretamente.

## Passo 1: Instalar Dependências

Antes de começar, você precisa instalar as dependências do projeto. Para isso, utilize o comando abaixo:

```sh
npm install
```

Esse comando irá baixar e instalar todos os pacotes necessários listados no arquivo package.json.

## Passo 2: Iniciar o Projeto

Depois de instalar as dependências, você pode iniciar o projeto utilizando o comando:

```sh
npx expo start
```

Este comando inicializará o servidor de desenvolvimento do Expo.
Você verá um QR code no terminal e uma nova janela do navegador será aberta com o Expo Dev Tools.

## Passo 3: Visualizar o Projeto no Celular

Para visualizar o projeto no seu dispositivo móvel, você precisará do aplicativo Expo Go. Para isso, baixe o APK do Expo 50 usando o link abaixo:

[Baixar Expo Go APK](https://d1ahtucjixef4r.cloudfront.net/Exponent-2.30.11.apk)

Passos para visualizar no celular:

1. Instale o Expo Go no seu dispositivo utilizando o APK fornecido.
2. Abra o aplicativo Expo Go.
3. Escaneie o QR code que apareceu no terminal ou na janela do Expo Dev Tools no navegador.

Seguindo esses passos, você conseguirá rodar e visualizar seu projeto Expo no seu dispositivo móvel.
